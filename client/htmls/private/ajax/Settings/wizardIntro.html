This wizard will help you to setup and configure Catalyst to manage DevOps assets across data center and cloud.

  • 1. Configuration Management: Configure your own chef server. You can add either a hosted chef server or an on-premise installation of chefserver.

  • 2. Provider Configuration: You can configure multiple cloud provider accounts of type Aws, Azure, OpenStack and VmWare within RLCatalyst.

  • 3. Repositories: You can configure Nexus repository, Docker Repository and Jenkins Server.

  • 4. Telemetry Configuration: Configure ELK, StackStorm and Sensu application for monitoring heath of the data center.

  • 5. CMDB Configuration: You can configure Configure ServiceNow ITMS application.
